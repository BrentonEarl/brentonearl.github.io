Urban Terror

11-16-2013 by Brent Earl

Category entertainment

Way back in the day I played Doom, Quake 2, Quake 3, Action Quake, Medal of Honor: Allied Assault, Medal of Honor: Breakthrough. I was a big fan of first person shooters. Out of nostalgia I decided to start playing Urban Terror because it is based on the code from Quake 3 Arena. Of course it has different player models and maps, but there are a few maps people recreated that I played in the previously mentioned first person shooters. I used to be very strict about the realism involved in the games I played, but that all went away with my long time binge of MMO's. Some aspects of Urban Terror are realistic but Urban Terror plays more like Quake 2, which was not realistic at all.

In favor of completely immersing myself in the game I decided to put up a game server for Urban Terror. The installation process and configuration is quite simple. Setting up the server is literally no more complicated than adding values to a series of 1 and 0 switches for each feature you would like the server to allow. I will be adding better maps and removing the bots as more players join to play on it consistently. I may also hack the server binary for Urban Terror so that I have more control over various administrative tasks some time in the future. The current status of the game server can be found below in the banner.

Gametracker Page - Exit Status One - TDM 4.2

In addition to hosting my own game server, I also found a nice bunch of people to play the game with. Most of the Urban Terror player base is European players, but I was lucky enough to be accepted into a North American clan called Not Even Trying. All of their server information (including TeamSpeak) can be found on their web site. They also have an IRC channel on QuakeNet, #noteventrying, in case you would like to check it out.
